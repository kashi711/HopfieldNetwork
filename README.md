# 概要
---
### MATLABで作ったホップフィールドネットワークのプログラム  
初期値画像から一番類似度の高い記憶画像(Image_*.bmp)を想起して書き出す

　
# 主なプログラムの構成
---

  - HopField.m　:　ホップフィールドネットワーク本体
  - copy_array.m　:　HopField.mで使う関数（与えられた配列を値を置き換えて返す）
  - Image_*.bmp　:　記憶させる画像データ
  - Init.bmp　:　初期画像データ
　

　
# 動かし方
---
```sh
>> HopField
```
MATLABでプログラム、画像データにパスを通す  
上のコマンドを打てば実行される
